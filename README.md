# DCI Contracts
Suite of contracts for ICO on EOSIO Blockchain in 3 phases A, B, C 

## Contracts
### Token
A token contract for transferring token (as per the ICO rate) during ICO on EOSIO Blockchains.

#### Action
* `create`
* `issue`
* `transfer`

#### Table
* `accounts`
* `stat`


### ICO

#### Action
* `deposit` (payable action)
* `disburse`

#### Table
* `fund`
* `rates`
